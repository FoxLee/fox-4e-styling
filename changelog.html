It won't be here forever, so please turn it off when you next clear your chat!

2023-12-13: [0.11] Item Card Updates

  • [Chat Styles] Updated power cards for items (game objects that are weapons, equipment, tools, backpacks, consumables or powers with Usage>Item) to match the styling used in later books (Mordenkainen's) and some later Dragon Magazines. I wasn't sure whether to use this style or the original—both are used often by the late 4e era—but ultimately I decided to go with the peach-coloured one because it involves less stark white, and therefore goes better with both light and dark mode UI.
  • [Chat Styles] Updated power cards for NPC traits and powers to follow their sheet style.
  • In both of the above cases, auto output is not an exact match; I can't do that without overriding the chat card template, which I would like to do but haven't worked out yet. However, with a little custom markup, the non-auto output can be basically perfect. To this end I have implemented "basics" and "subheading" classes for these chat cards, and a new compendium of Item Styling Examples. Please check their description fields in "Source HTML" mode if you want to use these styles!
  • [Chat Styles] Updated loot cards to use the header colour from treasure parcels. I thought it might be a good idea to differentiate them from mechanically useful items, and the maroon from treasure parcels is pleasantly distinct.

2023-03-17: [0.9.43] Condition/special resistances

Allows user management of condition & special resistances. Up until now these have been visible but not editable, so you could only add them by updating a creature's JSON via import. Now you can manage them on-sheet, like you'd expect.

Note this behaviour depends on the base system update to do the same, so it will only work correctly if you're running DnD4e 0.3.24 or later—but it should also fail harmlessly on earlier versions.
